# Android Studio Live Template

## How to install

### macOS
```
cd ~/Library/Application\ Support/Google/AndroidStudio4.1
git clone git@github.com:peerapongsam/android_studio_live_template.git templates
```

### Windows
```
cd %APPDATA%\Google\AndroidStudio4.1
git clone git@github.com:peerapongsam/android_studio_live_template.git templates
```
